Settings

The Character and Weapon Event cards show how many Primogems you'd need to reach a 5★. Genshin's hard pity is 90 (Character) / 80 (Weapon) — if you personally never push that far, set a buffer below to track to a lower pity instead. E.g. a buffer of 10 tracks to 80 on Character and 70 on Weapon.

Import Wishes fetches your wish history directly from HoYoverse's API. Browsers block that request unless it comes with the right headers, so a list of free CORS relay services is tried in order (after trying a direct request first) until one works. These free services occasionally change their rules, get rate-limited, or shut down — if imports suddenly start failing, add a working relay's URL below (one per line, ending in the query param it expects, e.g. ?url= or ?quest=) and it will be tried before the built-in list. Leave blank to just use the built-in list.

Win/loss on the Character and Weapon Event banners is worked out by checking each 5★ pull against the standard-banner picks below — pulled straight from your Characters and Weapons lists. Tap to toggle; new characters or weapons occasionally join the standard banner, so update this if a recent pull looks mis-labeled.

Import Wish History

Run the PowerShell command while Genshin Impact is open on your wish history screen, then paste the copied URL below. New wishes will be merged with anything already stored for this account — you can re-run this anytime.

Import Calendar

Open HoYoLAB's Genshin homepage while logged in, open your browser's DevTools → Network tab, filter by Fetch/XHR, refresh the page, and find the request to act_calendar. Copy its full JSON response and paste it below. Imports are merged in by name and date, so nothing already saved gets removed — re-import every patch to keep building your history.

Import Roster

Open HoYoLAB's Genshin homepage while logged in, open your browser's DevTools → Network tab, filter by Fetch/XHR, and find the request to api/character/list. Copy its full JSON response and paste it below. This is your actual in-game roster, so it's treated as the source of truth: it sets ownership and constellation level for every character it lists, corrects element/rarity/weapon if anything here was off, and adds any character you own that isn't in this list yet. Characters not present in the import are marked as not owned.

Import GOOD Data

Paste a GOOD-format export (the JSON format used by Genshin Optimizer, Akasha, and similar tools — starts with "format":"GOOD"). This updates both Characters and Weapons in one go: character constellation and ownership are set directly from the export (it's your account's real state). Weapon copies work differently — a weapon's refinement rank reflects how many total copies fed into it, so if the same weapon key appears more than once (e.g. two separate Dragon's Bane entries), their refinements are added together as the total copies owned. Weapon totals only ever merge upward: if wish history or a previous import already puts a weapon higher than what's in this file, that higher number is kept — pulls never decrease.
